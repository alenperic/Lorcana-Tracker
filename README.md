# Lorcana-Tracker
A web-based counter for the [Lorcana TCG game](https://www.disneylorcana.com), featuring interactive lore counters and a 3D animated dice roller. This project helps players keep track of their lore in the game with a clean, user-friendly interface. Includes features like player name inputs, a reset button, and an engaging 3D dice roll animation for added fun.

#### Features:
- **Interactive Lore Counters**: Easily increment or decrement lore counts for both players.
- **3D Animated Dice Roller**: Clickable 3D dice that spins and lands on a random number.
- **Player Name Inputs**: Enter and display player names to personalize the game experience.
- **Reset Button**: Quickly reset both counters to zero with a single click.
- **SEO Optimized**: Improved visibility on search engines with relevant meta tags.
- **Google Analytics Integrated**: Track visitor interactions and gather insights.

#### Technologies Used:
- HTML, CSS, JavaScript
- 3D CSS Animations
- Google Analytics
